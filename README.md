# Diabetic_Retinopathy-# Diabetic Retinopathy Detection

This project detects the stage of Diabetic Retinopathy from retinal fundus images using Deep Learning.

## Models
- Custom CNN
- ResNet50
- EfficientNetB2

## Dataset
APTOS 2019 Blindness Detection

## Tech Stack
- Python
- TensorFlow
- OpenCV
- NumPy
- Pandas
- Streamlit
